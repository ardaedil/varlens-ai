# Changelog

## 0.1.0

- Added greenfield VARLens monorepo.
- Added shared JSON Schema, TypeScript types, and Python Pydantic contracts.
- Added FastAPI service with analyze, health, and model-info endpoints.
- Added transient upload handling and typed API errors.
- Added deterministic stub inference and explanation generation.
- Added Next.js analysis workbench with upload, preview, result, and unsupported-scope states.
- Added docs, fixtures, CI workflow skeletons, and model/data scaffolding.
